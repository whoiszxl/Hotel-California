## 项目初始化学习

### 创建路由和页面跳转
1. 编辑`src/router/index.js`，通过配置`routes`可以配置好路由路径，路由需要跳转的组件
```js
export default new Router({
  routes: [
    {
      path: '/',
      name: 'Home',
      component: Home
    }, {
      path: '/list',
      name: 'List',
      component: List
    }
  ]
})
```
2. 通过`import Home from '@/pages/home/Home'`可以导入组件，组件的基本写法如下
```html
<template>
  <div>home</div>
</template>

<script>
export default {
  name: 'Home'
}
</script>

<style>

</style>
```

3. 通过运行指定的url路由地址就能直接访问配置好的组件了


### 多页面应用和单页面应用对比

#### 多页面应用
1. 页面跳转会返回一个新的html页面
2. 优点：首页时间快，SEO效果很好
3. 缺点：页面切换慢

#### 单页面应用
1. 页面跳转是通过js渲染的
2. 优点：页面切换快
3. 缺点：首次加载时候稍慢，SEO差
